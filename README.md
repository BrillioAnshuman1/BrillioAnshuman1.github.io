## Overview

This website represents a **development and testing environment** used to prototype and validate conversational commerce integrations. The purpose of this environment is to build, test, and demonstrate messaging-based shopping experiences that connect messaging platforms (such as WhatsApp) with an eCommerce storefront.

This is **not a public commercial storefront**. It is a controlled sandbox used internally for software development, feature validation, and technical demonstrations.

## Purpose of This Environment

This sandbox is used to test:

* Messaging platform integrations
* Interactive product messages
* Checkout redirection flows
* Payment workflow simulations
* Order creation APIs
* Commerce automation logic

All functionality is strictly for **development and evaluation purposes only**.
